About ASG Racing Server

Assetto Corsa Competizione public racing server

ASG Racing is a public Assetto Corsa Competizione server where drivers compete on popular GT3 tracks, improve their lap times and compare their statistics with other racers.

This page automatically publishes the server leaderboard including:

  • 🏁 number of races
  • 🥇 wins
  • 🏆 podium finishes
  • 📊 average finish position
  • ⚡ best laps

Statistics are generated automatically from ACC Dedicated Server result files. After each race the data is recalculated and published on the website.

How points are calculated

Points are awarded using a GT-style system:

  • 1st place — 25 points
  • 2nd place — 18 points
  • 3rd place — 15 points
  • 4th–10th — decreasing points

Drivers also receive 1 additional point for the fastest lap in race.

Best laps

The Best Laps table contains the fastest lap times recorded both in qualifying and in race sessions. This makes it easy to compare the outright pace of the drivers.
